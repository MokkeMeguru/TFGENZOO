logscale_factor¶

logscale_factor

Type

float

Note

  • initialize
    mean = mean(first_batch)
    var = variance(first-batch)
    logs = log(scale / sqrt(var)) / log-scale-factor
    bias = -mean
  • forward formula (forward only)
    logs = logs * log_scale_factor
    scale = exp(logs)
    z = (x + bias) * scale
